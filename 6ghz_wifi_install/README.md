# Orin wifi 6E (6GHz) installation

Included here is the built driver (`ifx-backports`) for an Orin Nano or Orin AGX running Jetpack 6.2. You should be able to follow the instructions in `orin_wifi_install.txt`. If that doesn't work, or you hate yourself, you can use `orin_wifi_kernel_build.txt` to build the kernel and backports folder, and flash the kernel onto a board (no different from the default Jetpack 6.2 kernel)
